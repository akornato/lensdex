This is a [React Native](https://reactnative.dev) app for [Lens Protocol](https://lens.xyz), a decentralized graph on [Polygon](https://polygon.technology) blockchain for social media profiles. The app has two screens, recommended profiles and single profile details.

Tech:

- [Expo](https://expo.dev)
- [Lens GraphQL API](https://api.lens.dev)
- [Apollo GraphQL](https://www.apollographql.com)
- [Magnus UI](https://magnus-ui.com)

## Local dev

- `yarn start` starts Expo dev server
- `yarn apollo-codegen` updates TypeScript files for Lens GraphQL API queries
